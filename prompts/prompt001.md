ウォークマン NW-Z1000シリーズ に搭載されていたビジュアライザーをwebアプリで再現したいと思います。以下の技術スタックでの実装例を提示してください。

[アプリケーションの概要]

タイトル: MP3ビジュアライザー

機能:

- 複数のMP3ファイルをファイル選択ボタンおよびドラッグアンドドロップで読み込み、内容を解析し、プレイリストに一覧表示する
- プレイリストで選択されたMP3ファイルを再生しながら、ビジュアライザーを表示する
- ウォークマン NW-Z1000シリーズ に搭載されていたビジュアライザーの8つのテーマを切り替えて表示する
- プレイリストにあるMP3ファイルをループ再生・1曲ループ再生・シャッフル再生できる

[使用技術]

フロントエンド: HTML, CSS, JavaScript

[ソースコード構成]

- index.html
- styles.css
- app.js

[デプロイ方法]

- GitHub Pagesを利用してデプロイする
- Reactを利用する場合は、ビルドして静的サイトとしてデプロイする

[その他要件]

- レスポンシブデザイン対応
- ユーザーフレンドリーなUI

[エラーハンドリングの想定]

- MP3ファイルの読み込みに失敗した場合のエラーメッセージ表示
- MP3ファイルの解析に失敗した場合のエラーメッセージ表示
